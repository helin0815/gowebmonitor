# goWebMonitor

服务治理引擎Go语言sdk

## 功能

- 平滑重启
- 健康检查探针
- pprof性能分析接口
- prometheus metrics
- Apollo注册中心
- 链路追踪（待实现）

## 安装

### required

- go1.16

```shell
go get github.com/helin0815/gowebmonitor
```

## 使用

http服务，详见：[ginapp](examples/ginapp)

grpc服务，详见：[grpcapp](examples/grpcapp)
